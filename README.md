# Walmart Sales Forecasting System

A machine learning-based sales forecasting system built using Python, XGBoost, Streamlit, and Plotly.

## Features

* Predict future weekly sales
* Interactive dashboard
* Store-wise forecasting
* Sales trend visualization
* Performance metrics (R², MAE, RMSE)

## Technologies Used

* Python
* XGBoost
* Streamlit
* Pandas
* Plotly

## Dataset

Walmart Store Sales Dataset

## Model Performance

* R² Score: 0.9807
* MAE: $54,877.79
* RMSE: $78,372.13

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
